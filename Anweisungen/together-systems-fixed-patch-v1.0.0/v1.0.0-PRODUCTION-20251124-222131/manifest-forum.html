						Dieses Offline‑Forum ist der Text‑Anker innerhalb eines größeren Hyperkommunikations‑Systems:
						Audio‑/Video‑Räume, geteilte Code‑ und Formel‑Snippets sowie strukturierte Daten können
						später an dieselben Beiträge andocken. Die Offline‑Datei bleibt dabei immer die

						signierenden Sync‑Dienst angebunden werden können.
